# WattStatus

*WattStatus* é uma plataforma (site e aplicativo móvel) para controle e monitoramento de consumo de energia elétrica, com foco em promover a sustentabilidade e o consumo consciente. Por meio de dados granulares por aparelho, gamificação e recomendações baseadas em IA, ajudamos usuários residenciais e corporativos a reduzir custos e minimizar impactos ambientais.

---

## 🔎 Visão Geral

- *Objetivo:* Auxiliar indivíduos e empresas a compreenderem seu padrão de consumo elétrico e adotar hábitos energeticamente eficientes.
- *Público-Alvo:*
  - Usuários residenciais que desejam ter clareza sobre sua conta de energia e identificar aparelhos mais consumidores.
  - Pessoas engajadas em sustentabilidade que buscam reduzir a pegada de carbono via consumo consciente.
- *Plataformas:*
  - *Site Web:* Dashboard responsivo, CMS de conteúdos e relatórios interativos.
  - *App Mobile (iOS/Android):* Acesso on-the-go, notificações em tempo real e recursos offline.

---

## 🚀 Funcionalidades Principais

1. *Monitoramento em Tempo Real*
2. *Análises e Previsões com IA*
3. *Gamificação & Educação*
4. *Marketplace & Financiamento*
5. *Compliance ESG & Relatórios Corporativos*

---

## 🛠️ Stack Tecnológica

- *Frontend Web:* Next.js, React, Tailwind CSS  
- *App Mobile:* React Native / Flutter  
- *Backend:* Node.js (Express) / Python (FastAPI)  
- *Banco de Dados:* PostgreSQL / MongoDB  
- *Mensageria:* MQTT / Kafka  
- *Infraestrutura:* AWS, Docker, Terraform  
- *CI/CD:* GitHub Actions, Fastlane
